# SIT708_3.1C

## Overview

This project is an Android Quiz App developed for SIT708 Task 3.1C. The app allows users to enter their name, answer quiz questions, view their final result, and start a new quiz.

The application uses multiple activities to separate the main screen, quiz screen, and result screen. It also includes light and dark mode colour design to improve the visual appearance and readability of the app.

## Features

- User name input
- Start quiz from the main screen
- Multiple-choice quiz questions
- Answer selection
- Score calculation
- Result screen
- Take a new quiz option
- Finish quiz option
- Light mode and dark mode colour design
- Simple multi-activity navigation
